This library defines numbers with units as *values*.

### Documentation

    ```xml
    <dependency>
      <groupId>org.tquadrat.library</groupId>
      <artifactId>org.tquadrat.foundation.value</artifactId>
      <version>0.1.0</version>
    </dependency>
    ```
